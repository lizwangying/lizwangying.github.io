二叉树入门

二叉树遍历分为： 二叉树深度优先遍历 ( DFS ，即为 Depth First Search ) 前序遍历 中序遍历 后序遍历 二叉树广度优先遍历 (BFS，即为 Breadth First Search ) 层序优先遍历 记忆心得：前中后是将根节点作为参照物，比如前序就是根节点第一个遍历，中序就是根节点第二个，后序就是根节点最后。 后文以此二叉树为例： 12345
